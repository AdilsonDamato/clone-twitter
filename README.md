## Prerequisites

- [Node LTS](https://nodejs.org/en/)
- [Yarn](https://classic.yarnpkg.com/pt-BR/)

## Installing

```bash

  # Clone repository
  $ git clone https://github.com/pereiradilson/clone-twitter.git

  # Enter folder
  $ cd clone-twitter

  # Install dependences
  $ yarn

  #Execute
  $ yarn start

```
